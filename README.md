# mi-slides
Slidedecks in Markdown mit Reveal.js im MI Style

## Voraussetzungen
MI-Slides braucht:
- NPM ab Version 6.0
- Node ab Version 9.0 

## Getting Started

**Repo lokal klonen:**

``` git clone git@github.com:cnoss/mi-slides.git ```

**ins Verzeichnis wechseln:**

``` cd mi-slides ```

**Module installieren:**

``` npm install ```

**MI-Slides starten:**

``` npm start ```

**Demo Präsentation auswählen**

**Fertig**
